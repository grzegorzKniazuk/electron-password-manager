How to use Main menu
build - Click to edit password generator settings
show_chart - Click to show the application usage statistics
backup - Click to backup/restore Your data
help_outline - Click to get help
info_outline - Click to get informations about application and author
settings - Click to edit application settings
Table
content_copy - Click to copy password
edit - Click to edit credentials
visibility - Click to show password
delete - Click to delete credentials
Bottom menu
flash_on - Click to generate and copy random password
add - Click to add new credentials
